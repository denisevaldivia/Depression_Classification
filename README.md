# Depression_Classification

El presente proyecto abarca la clasificación de la depresión en base a una serie de variables en torno a los hábitos alimenticios, de higiene de sueño, de estrés, antecedentes familiares y salud mental propia. De esta manera, tiene como objetivo discernir si el sujeto padece o no depresión.
